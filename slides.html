Apa itu PyQt5? 🤔

PyQt5 adalah Python binding untuk framework Qt yang memungkinkan kamu membuat aplikasi GUI cross-platform dengan Python.

Bayangkan seperti Lego — kamu tinggal susun widget-widget yang sudah disediakan, dan boom! Aplikasi GUI kamu jadi! 🎉

Qt sendiri adalah framework C++ yang sudah dipakai oleh perusahaan besar seperti Mercedes-Benz, AMD, VirtualBox, dan banyak lagi!

Python 3 Qt 5 Cross-Platform Open Source

Core 1: Pola 4 Langkah 📋

Siklus wajib setiap aplikasi PyQt5

1

Buat QApplication

Instance wajib yang mengelola seluruh aplikasi app = QApplication(sys.argv)

2

Buat Window/Widget

Buat jendela atau widget utama window = QMainWindow() atau window = QWidget()

3

Tampilkan Window

Tampilkan jendela ke layar window.show()

4

Jalankan Event Loop

Mulai loop utama aplikasi app.exec_()
